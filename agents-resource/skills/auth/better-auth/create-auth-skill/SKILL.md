---
name: create-auth-skill
description: Scaffold and implement authentication in TypeScript/JavaScript apps using Better Auth. Detect frameworks, configure database adapters, set up route handlers, add OAuth providers, and create auth UI pages. Use when users want to add login, sign-up, or authentication to a new or existing project with Better Auth.
---

This skill walks through a two-phase process: first gather requirements by scanning the project and asking structured questions (framework, database, auth methods, plugins, UI style), then implement based on the confirmed plan. It covers installation, server/client config, route handlers, database adapters, plugin wiring, UI patterns, and a security checklist.
