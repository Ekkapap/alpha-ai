Bun is an all-in-one JavaScript/TypeScript runtime with a native package manager, bundler, and test runner that is 10-100x faster than Node.js for installs and has ~25ms startup time. It runs TypeScript and JSX natively without a transpile step, provides built-in APIs for file I/O, HTTP serving, WebSockets, SQLite, and password hashing, and is largely compatible with Node.js APIs.
