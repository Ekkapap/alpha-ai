Tailwind CSS is a utility-first framework with mobile-first responsive breakpoints, built-in design tokens for colors/spacing/typography, and v4.1 CSS-first configuration via `@theme`. Use utility classes directly in markup, extract repeated patterns into components with CVA, and follow accessibility best practices (focus rings, ARIA labels, motion preferences).
