Tailwind CSS v4 uses CSS-first configuration via `@theme` blocks instead of `tailwind.config.ts`, OKLCH color tokens for perceptual uniformity, and `@custom-variant dark` for dark mode. Build components using CVA for type-safe variants, compound component patterns, and semantic design tokens; animate with native `@keyframes` in `@theme` and `@starting-style` instead of plugins.
