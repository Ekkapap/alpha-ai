Analyze UI screenshots to extract layout structure, components, colors, spacing, and responsive patterns, then generate complete production-ready code in the user's preferred framework (default: React + Tailwind CSS). Always match colors exactly, use semantic HTML, include accessibility attributes, and deliver all necessary files with usage instructions.
